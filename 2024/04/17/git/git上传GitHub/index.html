在 GitHub 上创建仓库：

首先，登录到你的 GitHub 账户。
点击页面右上角的加号图标，然后选择“New repository”创建一个新仓库。
给仓库起一个名称，并设置好其他选项（如描述、公开/私有等）。
完成设置后，点击“Create repository”按钮。

在本地初始化 Git 仓库：

感觉拉去了，应该就可以不执行了（跳过

打开命令行工具（如终端或命令提示符）。
使用 cd 命令导航到包含你要上传的文件的本地文件夹。
运行 git init 命令来初始化一个新的 Git 仓库。

将文件添加到本地仓库：

使用 git add <file> 命令将文件添加到 Git 暂存区。你也可以使用 git add . 将所有文件添加到暂存区。

提交文件到本地仓库：

运行 git commit -m “提交说明” 命令提交文件到本地仓库。替换 “提交说明” 为你的提交信息。

将本地仓库与 GitHub 仓库关联：

在 GitHub 仓库页面找到“Quick setup”部分的仓库地址（HTTPS 或 SSH）。
在命令行中运行 git remote add origin 命令，将本地仓库与 GitHub 仓库关联起来。替换 为你的 GitHub 仓库地址。

推送本地提交到 GitHub：

运行 git push -u origin master 命令，将本地提交推送到 GitHub 仓库。这会将你的本地 master 分支与 GitHub 仓库的 master 分支进行同步。如果你使用了其他分支名，将 master 替换为你的分支名。

git checkout -b <branch_name>
